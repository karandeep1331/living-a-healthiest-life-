Team Page

On the team page, where the user can learn about the team who created the website and the user can also find out which team member did which page in the website. And were you can learn more about the team, which includes the name and the roles that were assigned.

Technical Description

On the team page that I have created I have used HTML and CSS to build the page. Throughout the code I have used arrange of Div class, this so that I can change the style of the written text or box that is displayed on the page, As shown on the team page there is a box container which reveals about each team member when the user hover over it. I also made an DIV class for the details of the team member, were I can control the details size, colour and text aligning. On the webpage I have written that the user would need to hover over the image to learn more this is because user's may think that its only pictures on the team page not any information, when the information is hidden in a container. When the user hover each image the user will find out the editor name and the role that was carried out in the website.

the main CSS that I have used to create the team page on the website. The .team-container class uses display: flex; to align the team members in a row while center them with justify-content: center;. It also includes gap: 20px; to create spacing between members and margin: 110px auto; to ensure proper positioning on the page. Each team member is styled with the .team-member class, which gives them a white background, rounded corners (border-radius: 10px;), and smooth hover effects using transition: transform 0.3s, opacity 0.3s;. When hovered over, the .team-member scales slightly (transform: scale(1.05);) and reduces opacity (opacity: 0.8;) for a subtle interactive effect. The profile image is contained within .container, which ensures the image is rounded with border-radius: 100px; and fits properly using object-fit: cover;. Additionally, the .details class is initially hidden (display: none;), but when a user hovers over a team member, the details become visible (display: block;), creating an interactive and engaging experience.

Accessibility

On the team page I have created the webpage so that it can help users with accessibility, the first accessibility that I have added is alt text, this is so that the users have enhance accessibility for screen readers. Alt text is brief text description what the image is about. The second accessibility that I have added in the webpage is keyboard navigation, this is so that the users can make interactive thumbnail accessible via the keyboard.

Challenges and Lessons Learned

During the process of making the feedback page there was challenges and lessons that were learned. Such as I had to make all the fields that I created all required to fill, which means that the user wont be able to press the submit button without filling out the required fields. This was challenging to create, but however after doing research and looking at lectures I figured out how to make the fields required, so that an error message appears if the field is incorrect format, or invalid. I have now learned how to make the field have an error message appear for the users if the submit button is pressed.

The second challenge that I had during the process was styling error with using CSS, this is for when the fields on the feedback form is left empty, but should be filled. It was challenging to create red colour border on the field box for the user to know that the fields needs to be filled. However I did research and asked my team for support and I managed to make a styling colour border for the feedback form.

Creating the rating system for the feedback form was challenge to create. This I had to make the stars that are created interactive and hover effects, so that when the user clicks on the stars all the stars appear if its more than one star. With the use of CSS and research on the websites, I manage to use CSS to make hover effects and make the rating system interactive for the users to click on. I also made the colour of the stars to gold.

Making the team page there was few challenges that were occurring when creating the page, as the requirements that were given to me, was to make the information about the user hidden, were the user has to hover over the container, to see the information about the student and the role that the student did during the website. With the use of the container that I created, I managed to make the text in the box so that it's not visible on the page. I did have some challenges making that, but I used websites and video to help me create the container, and then I implemented more additional code, so that it meets the requirements.

on the content page there were few challenges that I was facing when I was creating the page. The first issue was that I had to think of a topic within healthy life and well-being which is our topic that we choice to make the website. Then after this I had issues with the images, on the page. As the pictures wasn't the same sizes the images was making the web page more bigger. But however I managed to sort the problem out.
